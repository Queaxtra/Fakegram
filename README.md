# Fakegram

Ready to create a fake post for Instagram?
## Source Code Setup

1) Download source codes.
```bash
https://github.com/Queaxtra/Fakegram
```

2) Install.

```bash
  npm install
```

3) Run the site.

```bash
  npm run dev
```
## Note
The "Create message" part of the site is NOT MOBILE COMPATIBLE!

Please do not forget to put it in "Desktop Mode" when creating a post.
It will be mobile compatible in future updates.
